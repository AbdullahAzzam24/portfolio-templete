Employee Manage Application

This application for manager in the corporate, this for maintain internal employee. The manager account added from the firebase authentication. The manager can add some employees by click add button and fill the employe form, the employee list will show after click save button. The manager can edit the employee by click one employee of the list. After choose the employee, the manager can edit shift and phone number. The manager can send a message to the employee after change the shift by click the Text Schedule button. The manager can fire the employee if don't satisfied with the employee by click Fire Employee button.
